⚡ Configuration Firebase

Avant de commencer, configure Firebase pour activer le live sync des votes.

1
Va sur console.firebase.google.com → Créer un projet → Nom de ton choix, désactive Google Analytics si tu veux.
2
Dans le projet : clique ⚙️ Paramètres du projet → onglet Général → Fais défiler jusqu'à "Vos applications" → clique </> Web → Enregistre l'app.
3
Copie la config qui apparaît (apiKey, authDomain, etc.) et colle-la ci-dessous.
4
Dans Firebase : Build → Realtime Database → Créer une base → Mode test → Ensuite Authentification → Sign-in method → Anonyme → Activer.

Le mode démo stocke tout en local — pas de sync entre appareils, mais utile pour tester.
